# 🚀 YouTube Playlist Scheduler

🔗 Live Demo: https://playlist-scheduler-8uy7-e78jpnkfg.vercel.app/

## 📌 Overview
The YouTube Playlist Scheduler is a web application that converts long YouTube playlists into structured, time-based daily schedules.

Users can input a playlist and define how much time they want to spend daily, and the app automatically distributes videos across days efficiently.

---

## ✨ Features
- ⏱️ Generate schedule based on daily watch duration
- 📅 Smart distribution of videos across multiple days
- 🌙 Dark / Light mode toggle
- 📊 Export schedule to CSV & PDF
- ⚡ Smooth UI with animations and loading indicators

---

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, Bootstrap
- **Scripting:** JavaScript
- **Deployment:** Vercel

---

## ⚙️ How It Works
1. User inputs a YouTube playlist URL
2. Enters daily watch time (in minutes)
3. App calculates total duration
4. Distributes videos into optimal daily chunks
5. Outputs structured schedule
